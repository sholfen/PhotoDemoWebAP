# 新刊預購表單

專案啟動前，務必將 demo.bak 檔案還原到 localdb (版本為 V.16)，確定 demo 這個資料庫後，再確認是否有下列下張資料表：

- Product
- ProductOrder
- BundleProduct
- GroupOrder

專案為 ASP.NET，版本 .NET6。

專案啟動後，可以在下列網址測試新刊預購：

`https://[FQDN]/Home/Product`

新增後，可到下列網址瀏覽所有訂單，也可以刪除或者修改聯絡資料

`https://[FQDN]/Home/OrderList`
